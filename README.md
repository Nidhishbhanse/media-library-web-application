# Media Library Web Application

Overview
This project is a web-based media library application developed as part of an academic coursework project. The application was designed to run on a controlled local network environment without external internet access.

Key Features
- Browse and play audio content through a locally hosted interface
- Organized media structure for easy navigation
- Simple and clean user interface
- Optimized for tablet-based access (used with iPads in a local network)

Technical Details
- Frontend: HTML, CSS, JavaScript
- Deployment: Local network (offline environment)
- Focus areas: UI design, usability, and structured content management

Project Context
This project was developed under real-world constraints where internet access was restricted. The goal was to build a functional and user-friendly system that operates entirely within a closed network environment.

Notes
Media files (audio and images) are not included in this repository due to size limitations. The repository focuses on the application structure and source code.
